Lab 13 - Game Players API & Website Connection

How to run:
1. Open this project folder in VS Code.
2. Open terminal in this folder.
3. Run: npm install
4. Run: npm start
5. Open Players.html in browser to see players loaded from API.
6. Open Schedule.html in browser to see tournaments and teams loaded from API.

API test links:
http://localhost:3000/
http://localhost:3000/players
http://localhost:3000/players?game=PUBG
http://localhost:3000/tournaments
http://localhost:3000/teams

Lab requirements completed:
- GET /players API
- GET /players?game=PUBG filter API
- GET /tournaments API
- GET /teams API
- Frontend fetches players from backend API
- Schedule page fetches tournaments and teams from backend API
